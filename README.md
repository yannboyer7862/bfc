This is the repository of bfc, a small, dumb and basic Brainfuck interpreter written in C++.



NOTE : If you are using an x86_64 microprocessor and Linux, please, DO NOT use Clang as your compiler.

(This is because the interpreter is almost 2 times slower when compiled with Clang, it's faster with GCC)
